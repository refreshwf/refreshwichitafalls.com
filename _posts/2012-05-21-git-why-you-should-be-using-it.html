Jorin Slaybaugh gives you a run down on git and why it is important that you use version control on every project. He will walk you through the basics of creating repos, making commits and pulling. Whether on a team or working solo, git is a resource everybody should have in their tool belt.

Jorin graduated from Midwestern State University and is currently the lead developer and VP of Technology for E-vent Software. He is a .NET guy with a love of jQuery and using MVC to make API's for single-page JavaScript web apps. You can find his blog at cacheandquery.com, follow him on twitter or browse his repositories on github.
