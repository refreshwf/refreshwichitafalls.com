Create a page with four colored rectangles of equal size that form a quadrant (each is adjacent to exactly two other squares two sides). Those four rectangles must collectively fill the browser window regardless of the size of the browser window. For an example, please view this image. Colors are up to you.

BONUS
An extra entry will be awarded for an entry which not only meets the criteria but also adjusts when the browser is resized.

BONUS
An extra entry will be awarded for an entry which meets the criteria solely with HTML+CSS and uses NO JavaScript.

Got the answer? Create a CodePen or a JS Fiddle and E-mail the answer.
All answers are due by the next meeting and must be done in JavaScript + HTML + CSS.
