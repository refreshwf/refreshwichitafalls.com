About

Code challenges are a fun way for participants to get involved each month and increase their chances to win prizes at each meeting.

Rules

Code challenges are posted in advance of a meeting and are due by the start of the meeting to hello@refreshwichitafalls.com. Since we are a group of web professionals all solutions must be done in JavaScript + HTML + CSS using a tool like CodePen or JS Fiddle.
